## Installation

1. Download, and Execute "asteroids.exe" and enjoy!
2. Or download source
3. Load the asteroids.sln
4. In the Project-Settings
5. with ..._ALL -> all DLLs are including in the exe

## Usage

| Action               | Keyboard| Mouse        |
|----------------------|---------|--------------|
| speed++              | UP      | scroll up    |
| speed--              | DOWN    | scroll down  |
| angle++              | RIGHT   | Point        |
| angle--              | LEFT    | Point        |
| fire                 | SPACE   | left button  |
| shild                | CONTROL | right button |
| stop (Only with item)| SHIFT   | middle button|

With Setup (F3), in the Start-Screen, you can change
